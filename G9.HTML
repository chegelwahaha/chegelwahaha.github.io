课程理念 | Educational Philosophy

本课程以“计算思维、数据思维、创新实践”为核心，强调信息技术与学科知识的深度融合，培养学生利用信息技术解决复杂问题的能力。进一步提升学生的编程能力和数据分析能力，并引入人工智能的前沿概念和应用。课程注重深度学习、数据驱动、创新实践、跨界融合、面向未来和个性发展。

This course focuses on "computational thinking, data thinking, and innovative practice," emphasizing the deep integration of information technology and subject knowledge to cultivate students' ability to solve complex problems using IT. It further enhances students' programming and data analysis skills, and introduces cutting-edge concepts and applications of artificial intelligence. The course emphasizes in-depth learning, data-driven approaches, innovative practice, interdisciplinary integration, future-oriented learning, and personalized development.

指导原则 | Guiding Principles

本课程遵循中国教育部《义务教育信息科技课程标准（2022年版）》9年级的主要内容、美国CSTA K-12计算机科学标准以及ISTE学生标准。

This course follows the main content of the 9th grade of the "Compulsory Education Information Technology Curriculum Standards (2022 Edition)" of the Chinese Ministry of Education, the American CSTA K-12 Computer Science Standards, and the ISTE Standards for Students.

课程设置设计的思想 | Course Design Philosophy

本课程采用“项目式学习”（Project-Based Learning）、“探究式学习”（Inquiry-Based Learning）、“基于设计的学习”（Design-Based Learning）和“协作式学习”（Collaborative Learning）的设计思想。

This course utilizes the design principles of "Project-Based Learning," "Inquiry-Based Learning," "Design-Based Learning," and "Collaborative Learning."

课程内容 | Course Content

本课程主要包含以下内容：

  • HTML+JS游戏设计与实现 (HTML+JS Game Design and Implementation)
  • Microsoft Office回顾与数据可视化 (Microsoft Office Review and Data Visualization)
  • Python编程进阶 (Advanced Python Programming): 学习Python的函数、模块、文件操作、面向对象编程等。(Learn Python functions, modules, file operations, object-oriented programming, etc.)
  • 3D建模与打印 (3D Modeling and Printing)
  • AI与机器学习 (AI and Machine Learning): 学习AI基本概念，着重介绍机器学习和算法的概念。(Learn basic AI concepts, with a focus on the concepts of machine learning and algorithms.)
  • Python图形界面编程与项目实践 (Python GUI Programming and Project Practice): 学习使用Python的Tkinter库进行图形界面编程。(Learn to use Python's Tkinter library for graphical user interface programming.)

评估方式 | Assessment Methods

本课程采用多元化的评估方式，包括课堂观察、作品评价、互评与自评、随堂测验、项目报告/展示和代码审查和测试。

This course uses a variety of assessment methods, including classroom observation, project evaluation, peer and self-assessment, quizzes, project reports/presentations, code review, and testing.
